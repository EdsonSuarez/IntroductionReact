# IntroductionReact
React exercises

## Install React step 1

```
# Windows
md react-recipes && cd react-recipes
md src
md public
touch package.json
echo "{}" > package.json

# Linux or macOS
mkdir react-recipes && cd react-recipes
mkdir src
mkdir public
touch package.json
echo "{}" > package.json
```

## Install React step 2

```
npm install --save-dev snowpack
npm install react react-dom
```
